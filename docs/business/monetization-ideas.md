# 과금 아이디어

## 방향

나무 장터는 일반 쇼핑몰보다 B2B 조경수 매칭 플랫폼에 가깝다. 그래서 초기 수익화는 결제 수수료만 보는 것보다 `수요 등록 -> 매칭 -> 문의/견적 -> 거래 성사` 흐름에서 돈을 받는 구조가 현실적이다.

우선순위는 다음과 같다.

1. 공급자 구독료
2. 문의/견적 리드 과금
3. 상단 노출/광고
4. 거래 성사 수수료
5. 운송/식재/현장 검수 중개
6. 가격 데이터/시세 리포트
7. 농장 재고관리 SaaS

## 1. 공급자 월 구독료

가장 현실적인 1순위 모델이다. 조경수 공급자, 농장, 유통업체가 재고를 등록하고 수요자 문의를 받는 구조이므로 공급자에게 월 이용료를 받을 수 있다.

| 플랜 | 월 과금 | 제공 기능 |
|---|---:|---|
| 무료 | 0원 | 재고 10개 등록, 사진 3장, 문의 월 3건 |
| 베이직 | 29,000원 | 재고 100개, 문의 무제한 |
| 프로 | 79,000원 | 상단 노출, 지역 우선 매칭, 견적서 다운로드 |
| 프리미엄 | 150,000원 이상 | 대량 재고 업로드, 영업 리포트, 담당자 지원 |

초기에는 공급자가 돈을 내기 꺼릴 수 있으므로 `무료 등록 + 문의/노출 제한` 방식이 좋다.

## 2. 문의/견적 리드 과금

수요자가 실제 구매 의사가 있는 글을 올리면, 공급자가 그 문의를 열람하거나 견적을 보내는 순간 과금하는 모델이다.

| 과금 지점 | 금액 예시 |
|---|---:|
| 문의 열람 1건 | 1,000~3,000원 |
| 견적 참여 1건 | 3,000~10,000원 |
| 대형 공사/공공 발주 리드 | 30,000원 이상 |

B2B 마켓플레이스에서는 qualified lead를 판매하는 CPL 모델이 자주 쓰인다. 이 앱은 이미 매칭과 문의 흐름이 있으므로 기능 변경 없이 붙이기 쉽다.

참고:

- Founder Institute, marketplace monetization: https://fi.co/insight/five-ways-to-monetize-your-marketplace-business

## 3. 거래 성사 수수료

거래금액의 일정 비율이나 건당 정액을 받는 방식이다.

| 방식 | 금액 예시 |
|---|---:|
| 거래금액 비율 | 1~3% |
| 건당 정액 | 30,000~100,000원 |

다만 조경수 거래는 현장 확인, 운송, 수형 품질, 하자 문제 때문에 플랫폼 밖에서 거래가 빠질 가능성이 있다. 따라서 초반부터 수수료만 노리는 것보다 구독료/리드 과금으로 시작하고, 나중에 결제, 계약, 운송까지 붙인 뒤 거래 수수료를 붙이는 순서가 좋다.

## 4. 상단 노출/광고 상품

공급자가 자기 재고와 농장을 더 잘 보이게 하는 유료 노출 모델이다.

- 지역 검색 상단 노출
- 수종별 추천 공급자
- 급매/대량 재고 배너
- 메인 화면 추천 농장
- 특정 규격 검색 결과 우선 노출

기존 조경수 거래 플랫폼과 협회 사이트도 매물/회원사 정보 노출 구조를 갖고 있다. 공급자는 신규 거래처 확보를 위해 노출 비용을 낼 가능성이 있다.

참고:

- 한국조경수협회: https://www.klta.or.kr/
- 트리디비 앱: https://play.google.com/store/apps/details?id=kr.co.treedb

## 5. 가격 데이터/시세 리포트

플랫폼이 계속 거래 데이터를 쌓으면 실거래 기반 조경수 시세 정보를 만들 수 있다.

- 월간 조경수 시세 리포트
- 지역별 수종 가격 비교
- 규격별 평균 단가
- 수요 급증 수종 알림
- 설계/견적 참고 단가 API
- 공급자용 내 재고 가격 포지션 분석

조경수는 규격, 지역, 계절, 운송 조건에 따라 가격 편차가 크다. 실거래 데이터가 쌓이면 단순 매칭보다 더 높은 부가가치를 만들 수 있다.

참고:

- 산림청 임산물유통정보시스템 조경수 가격 정보: https://www.forestinfo.or.kr/public/tree_prinfo/selectTreePrinfoFoppList.do
- 조경수 가격 정보 관련 기사: https://www.latimes.kr/news/articleView.html?idxno=52981

## 6. 운송/식재/현장 검수 중개

조경수 거래는 물건 구매만으로 끝나지 않는다. 굴취, 상차, 운송, 하차, 식재, 하자 관리가 뒤따른다.

- 운송업체 연결 수수료
- 식재팀 연결 수수료
- 현장 검수 대행
- 사진/영상 검수 리포트
- 하자 보증 옵션
- 공사 일정 관리 수수료

이 모델은 MVP 이후 붙이는 고부가 모델로 적합하다.

## 7. 농장 재고관리 SaaS

공급자에게 거래 플랫폼이 아니라 `내 농장 재고관리 프로그램`으로 판매할 수 있다.

- 농장별 구역 관리
- 수목 QR 코드
- 사진 이력 관리
- 굴취 가능 시기 관리
- 거래처별 견적서
- 엑셀 업로드/다운로드
- 재고 변동 이력
- 수종/규격별 판매 리포트

이 방향은 단순 중개보다 락인이 강하고, 장기 구독 모델로 확장하기 좋다.

참고:

- 조경수 거래/관리 솔루션 관련 기사: https://plus.hankyung.com/apps/newsinside.view?aid=202512205642d

## 추천 실행 순서

### 1단계: MVP 수익화

```text
무료 등록 + 공급자 유료 구독
문의 열람/견적 참여 포인트
상단 노출 광고
```

### 2단계: 거래 지원

```text
거래 성사 수수료
운송/식재 중개
실거래 시세 리포트
```

### 3단계: SaaS 확장

```text
농장 재고관리 SaaS
QR 수목 이력 관리
공공/설계사용 가격 데이터 API
```

## 현재 앱 기준으로 가장 먼저 붙일 과금 포인트

현재 앱은 이미 `수요 등록 -> 매칭 -> 문의 요청` 구조를 갖고 있다. 그래서 가장 먼저 붙이기 쉬운 과금 포인트는 다음 두 가지다.

1. 공급자 구독
2. 문의 리드 과금

초기 가설:

```text
공급자는 무료로 재고를 등록할 수 있다.
하지만 문의를 일정 건수 이상 받거나, 상세 연락처를 보려면 유료 플랜 또는 포인트가 필요하다.
```

이 방식은 결제/정산/거래 추적을 모두 만들기 전에도 실험할 수 있다.
