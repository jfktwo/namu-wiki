# 디자인 개편 실행 계획

생성 시각: 2026-06-14T05:37:04.531Z

## 목표

나무 장터를 감성적인 모바일 앱 느낌에서 신뢰감 있는 조경 B2B 거래 도구로 정리한다.

## 1차 개편 범위

- 공통 컴포넌트: Button, Card, Badge, Input
- 목록 카드: 재고, 수요, 매칭
- 인증 화면: 로그인, 회원가입
- 문구: 깨진 한글 복구

## 단계별 작업

### 1단계: 문구/기초 정리

- [ ] 깨진 한글 문자열 복구
- [ ] 페이지 제목/버튼/오류 메시지 문구 통일
- [ ] 상태 문구 사전 작성

### 2단계: 디자인 토큰 정리

- [ ] 색상 토큰 정리
- [ ] radius 8px 기준 적용
- [ ] shadow 사용 최소화
- [ ] focus/disabled/empty/error 상태 정의

### 3단계: 카드 재설계

- [ ] 재고 카드: 사진 -> 수종 -> 규격 -> 수량 -> 위치/거리 -> 가격 -> 액션
- [ ] 수요 카드: 수종 -> 규격 -> 필요 수량 -> 납품 주소 -> 희망 시기 -> 예산 -> 매칭 후보
- [ ] 매칭 카드: 거리/수종/규격/수량/가격 점수 분리

### 4단계: 모바일/데스크톱 대응

- [ ] 모바일 하단 탭 유지
- [ ] 데스크톱은 목록 폭 확대 또는 2열 레이아웃 검토
- [ ] 관리자 화면은 테이블 중심으로 별도 정리

### 5단계: 재캡쳐와 비교

- [ ] 같은 캡쳐 스크립트로 전/후 비교
- [ ] 화면별 개선 체크리스트 갱신
- [ ] 위키 문서에 변경 전/후 스크린샷 연결
